Directions for the Three Resistor Calculator tool

You can use this program to calculate the series or parallel values
to ensure the student is getting the correct answers.

  • All values will be changed to positive ones for calculations.
  • You must select a battery voltage and at least one resistor value.
  • If you use zero for the battery voltage, the results will not be displayed.
  • If you use zero for each resistor, the results will not be displayed.

If you experience calculation errors, press the
 Reset Values  button before a new calculation.

Keep up the good work!
